Коридор имеет размер 3xN. Для заданного N необходимо определить число различных способов уложить паркет в этом коридоре плитками размером 1x2 и 2x2 (сами варианты укладки паркета определять не надо).

Формат входного файла
Единственная строка содержит целое число N (1<=N<=40).
Формат выходного файла
Выведите число различных вариантов укладки паркета.

Пример
in.txt
2
out.txt
5
